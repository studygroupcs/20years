Bootstrap 3.2 New + Sass

This is a quick and simple setup for using Bootstrap-Sass in Mixture.

It currently uses Bootstrap-Sass v3.2.0, and has Autoprefixer activated.

Please note the project is in debugg mode by default. To change it, go to mixture.json and change "debug": true" to "debug": false".

Updated for Mixture v2.55 Bootstrap 3.2

  • Add javascript in assets/javascripts/application.js
  • Edit styles in assets/stylesheets/style.scss
  • Edit MEDIAQUERIES in assets/stylesheets/_mediaqueries.scss
  • Edit variables in assets/stylesheets/_variables.scss (Own and Bootstrap)
  • Add images in app/img
  • Add fonts in app/fonts
